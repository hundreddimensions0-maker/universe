## 📌 About

This repository contains Python-based animations covering core topics in physics and mathematics. Every animation is built from scratch using Manim CE, with a focus on making complex concepts visually intuitive — no camera, no mic, just pure visual storytelling.

Content is part of **The Light Series** and related educational projects published on YouTube and Instagram.

---

## 🗂️ Topics Covered

| Topic | Animations |
|---|---|
| ⚡ Electromagnetism | KCL Circuit, Faraday's Law, Bar Magnet Field Lines, Gauss's Law |
| 🌡️ Thermodynamics | Entropy, Blackbody Radiation |
| 🔬 Quantum Mechanics | Quantum Tunneling, Photoelectric Effect |
| 🔭 Classical Mechanics | Principle of Least Action |
| 💡 Light Series | EM Wave from Oscillating Electron |


## ▶️ Usage

```bash
# Render any scene
python -m manim electromagnetism/kcl_circuit.py KCLScene -pql

# Flags:
# -p  → preview after rendering
# -ql → low quality (fast)
# -qh → high quality (1080p)
```
